## 🔧 常见问题

### 问题 1: 服务器连接失败

**错误**: `Error reading SSH protocol banner` 或 `Connection closed by remote host`

**解决方法**:
1. 检查服务器是否在线: `ping play.devc.me`
2. 检查 SSH 端口: `nc -zv play.devc.me 22`
3. 等待几分钟后重试
4. 检查服务器 SSH 服务: 联系服务器管理员

### 问题 2: 服务启动失败

**检查日志**:
```bash
python3 remote_exec.py exec "tail -100 /tmp/backend.log"
```

**常见原因**:
- Python 依赖缺失: 需要安装 `websockets` 库
- 端口被占用: 检查是否有其他进程占用 8000 端口
- 语法错误: 检查上传的文件是否完整

**解决方法**:
```bash
# 安装缺失的依赖
python3 remote_exec.py exec "cd /home/zhuang/info-tech/backend && ./venv/bin/pip install websockets"

# 检查端口占用
python3 remote_exec.py exec "lsof -i:8000"

# 强制杀掉占用端口的进程
python3 remote_exec.py exec "fuser -k 8000/tcp"
```

### 问题 3: WebSocket 连接失败

**检查**:
1. 后端服务是否正常运行
2. 防火墙是否开放 8000 端口
3. WebSocket URL 是否正确

**测试命令**:
```bash
# 测试 WebSocket 端点（使用 websocat 工具）
# 如果没有安装: brew install websocat
echo "test" | websocat ws://play.devc.me:8000/ws/realtime/stream?device_id=test
```

### 问题 4: 腾讯云 ASR 连接失败

**检查环境变量**:
```bash
python3 remote_exec.py exec "cd /home/zhuang/info-tech/backend && cat .env | grep TENCENT"
```

**确保配置了**:
```
TENCENT_SECRET_ID=your_secret_id
TENCENT_SECRET_KEY=your_secret_key
```
