# Web UI 快速预览说明

Fork 项目后，想**不跑服务、不改代码**先看一眼界面，可以用下面两种方式之一。

---

## 方式一：直接打开静态预览页（最便捷）

1. 在仓库根目录找到 **`ui-preview.html`**
2. 用浏览器打开该文件即可：
   - **方式 A**：双击 `ui-preview.html`
   - **方式 B**：浏览器菜单「文件 → 打开文件」选择 `ui-preview.html`
   - **方式 C**：把 `ui-preview.html` 拖进浏览器窗口

**说明**：这是单文件静态页，展示左侧导航 + 顶栏 + 仪表盘式 KPI 卡片的布局与设计风格，**无接口、无登录、无真实数据**，仅用于确认 UI 风格和版式。

---

## 方式二：运行完整前端（联调 / 功能测试）

需要本地已安装 Node.js（建议 18+）。

```bash
cd frontend
npm install
npm run dev
```

浏览器访问 **http://localhost:5173**。  
登录、会话、上传、实时监听等需配合后端 API 使用，后端启动方式见仓库根目录 README。

---

## 小结

| 目的           | 推荐方式                     |
|----------------|------------------------------|
| 只看一眼 UI   | 直接打开根目录 `ui-preview.html` |
| 完整功能试用   | `cd frontend && npm i && npm run dev`，并启动后端 |

`ui-preview.html` 不依赖现有业务代码，仅作展示，可随时删除或忽略。
