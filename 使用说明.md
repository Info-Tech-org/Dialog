# 家庭情绪交互系统 - 使用说明

## 🚀 快速启动

### 方式一：一键启动（推荐）

双击运行 **`start.bat`**，脚本会自动：
1. ✅ 检查 Python 和 Node.js 环境
2. ✅ 创建必要的数据目录
3. ✅ 启动后端服务 (http://localhost:8000)
4. ✅ 启动前端服务 (http://localhost:3000)
5. ✅ 自动打开浏览器

### 方式二：手动启动

**启动后端：**
```bash
cd backend
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

**启动前端：**
```bash
cd frontend
npm run dev
```

## 🛑 停止服务

### 方式一：一键停止（推荐）
双击运行 **`stop.bat`**

### 方式二：手动停止
关闭后端和前端的命令行窗口

## 🔧 配置说明

### 腾讯云 COS 配置

配置文件：`backend/config/settings.py`

```python
# COS 配置
tencent_cos_region: Optional[str] = "ap-guangzhou"
tencent_cos_bucket: Optional[str] = "info-tech-test-1314143047"
tencent_cos_use_presigned_url: bool = True  # 使用预签名URL
```

**当前状态：**
- ✅ 已启用 COS
- ✅ 存储桶：`info-tech-test-1314143047`
- ✅ 地域：广州 (ap-guangzhou)
- ✅ 使用预签名 URL（私有读写，更安全）

### API 密钥配置

在 `backend/config/settings.py` 中配置：

```python
# 腾讯云 ASR
tencent_secret_id: str = "YOUR_SECRET_ID"
tencent_secret_key: str = "YOUR_SECRET_KEY"

# OpenRouter (LLM)
openrouter_api_key: str = "YOUR_API_KEY"
```

## 📝 API 端点

### 认证相关
- `POST /api/auth/login` - 用户登录
- `POST /api/auth/register` - 用户注册
- `GET /api/auth/me` - 获取当前用户信息

### 会话管理
- `GET /api/sessions` - 获取会话列表
- `GET /api/sessions/{id}` - 获取会话详情

### 音频上传
- `POST /api/audio/upload` - 上传音频文件
- `GET /api/audio/upload/status/{id}` - 查询处理状态

## 🐛 常见问题

### Q: 启动失败，提示"未找到 Python"
**A:** 请先安装 Python 3.11+，并确保添加到系统 PATH

### Q: 前端无法访问，显示空白页
**A:**
1. 检查 Node.js 是否安装
2. 运行 `cd frontend && npm install` 安装依赖
3. 确保端口 3000 未被占用

### Q: 后端启动失败，提示模块不存在
**A:** 运行 `cd backend && python -m pip install -r requirements.txt` 安装依赖

### Q: 上传音频后显示"占位数据"
**A:** 检查 COS 配置：
1. 确认 `tencent_cos_bucket` 和 `tencent_cos_region` 已配置
2. 确认 Secret ID 和 Secret Key 正确
3. 查看后端日志的详细错误信息

### Q: 登录失败
**A:**
1. 确认使用测试账号：`admin` / `admin123`
2. 检查后端是否正常运行
3. 查看浏览器控制台的网络请求

### Q: 如何创建新用户？
**A:** 运行 `cd backend && python reset_and_create_user.py`
